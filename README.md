# Agent Development Standard

> AIエージェントと人間が同じ品質基準でソフトウェア開発を行うための標準リポジトリ

## 概要

Agent Development Standard は、AIエージェントを活用したソフトウェア開発において、一貫した設計・実装・レビューを実現するための開発標準です。

本リポジトリは特定のプログラミング言語やフレームワークに依存しません。

また、Claude Code、Codex、Cursor、Gemini など特定のAIエージェントにも依存せず、共通して利用できる標準を提供することを目的としています。

---

## このリポジトリが提供するもの

- AIエージェント共通の行動規範
- ソフトウェア開発標準
- 設計ガイドライン
- コーディング標準
- プロンプトテンプレート
- ドキュメントテンプレート
- ベストプラクティス

---

## このリポジトリが提供しないもの

本リポジトリには以下は含まれません。

- アプリケーションのソースコード
- フレームワーク固有の実装
- ライブラリ
- Webアプリケーションの雛形

これらは別リポジトリで管理します。

## リポジトリの位置付け

本リポジトリは次の3層構造の最上位に位置します。

```
agent-dev-standard
        │
        ▼
webapp-template
        │
        ▼
your-project
```

### agent-dev-standard

共通で利用する開発標準を管理します。

### webapp-template

開発標準を適用したプロジェクトテンプレートです。

### your-project

実際のWebアプリケーションを実装するリポジトリです。

---

## 利用方法

1. 本リポジトリの内容を理解する
2. `webapp-template` に標準を適用する
3. 新しいプロジェクトをテンプレートから作成する
4. 必要に応じてプロジェクト固有のルールを追加する

---

## 基本方針

- 標準はプロンプトより価値がある
- AIではなくプロジェクトが品質を決定する
- 会話ではなく知識を残す
- 小さく改善し続ける
- 保守性を最優先に考える

詳細は `VISION.md` を参照してください。
